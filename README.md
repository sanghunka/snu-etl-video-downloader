# SNU eTL Video Downloader

Download lecture videos from [SNU eTL](https://myetl.snu.ac.kr).

[한국어](README.ko.md)

![screenshot](assets/screenshot.en.1280x800.png)

## Install

[Chrome Web Store](https://chromewebstore.google.com/detail/ajmkfkhbjinikeaihpakpaglfgdefehi)

## Usage

1. Go to a lecture video page on SNU eTL
2. Click the extension icon
3. The download dialog opens — confirm to save
